# dingtalk-videomeeting 创建视频会议

## 配置

将以下 JSON 保存为 `mcporter.json`，放到技能目录根路径：

```json
{
  "mcpServers": {
    "钉钉视频会议": {
      "type": "streamable-http",
      "url": "https://mcp-gw.dingtalk.com/server/<serverId>?key=<key>"
    }
  }
}
```
